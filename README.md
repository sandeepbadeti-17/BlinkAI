# BlinkAI ⚡
**Instant AI help while browsing — no tab switching, no copy-paste.**

BlinkAI is a lightweight Chrome extension that brings AI directly into your browsing experience. Select text on any webpage and instantly get explanations, summaries, answers, and follow-up help without leaving the page.

The goal is simple: **make AI feel native to the web.**

---

## ✨ Why BlinkAI?

Most AI workflows interrupt focus.

Typical workflow today:
1. Copy text  
2. Open ChatGPT or Gemini  
3. Paste content  
4. Ask the question  
5. Switch back to the page  

BlinkAI removes this friction completely.

Just **select → ask → continue reading.**

Perfect for:
- Learning while reading articles  
- Research and quick fact-checking  
- Email understanding (Gmail supported)  
- Productivity and daily browsing  
- Instant summaries and explanations

## 📦 Installation (Developer Mode)

1. Clone or download this repository

2. Open Chrome and go to:
chrome://extensions/


3. Enable **Developer Mode** (top-right toggle)

4. Click **Load unpacked**

5. Select the `dist` folder

The extension will now appear in your Chrome toolbar 🎉

---

## 🔑 Setup — Gemini API Key

1. Get a free API key from **Google AI Studio**

2. Open the BlinkAI extension popup

3. Paste your API key and click **Save Key**

You are ready to use BlinkAI ✨
